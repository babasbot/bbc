# bbc

**bbc** is a simple calculator language with interactive execution of 
statements.

## Getting Started

```
$ make
```

And then execute:

```
$ make install
```

## Examples

### Interactive Execution

```
$ bbc
1 * 2 + 3
5
```

### Pipe It! 

```
$ echo "(1 + 2) * 3" | bbc
9
```
